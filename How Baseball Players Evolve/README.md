# ⚾ MLB SQL Analysis

This project analyzes historical baseball player data using SQL to explore how player development, salaries, careers, and attributes have evolved over time.

The goal is to transform raw relational data into meaningful insights that can support decision-making in scouting, salary strategy, and player development.

---

## 📊 Project Structure

The analysis is organized into four key areas:

### 🏫 School Analysis
- Examines talent pipelines and player origins  
- Identifies top schools producing professional players  
- Analyzes how these pipelines evolved over time  

### 💰 Salary Analysis
- Explores team spending behavior  
- Analyzes salary trends across years  
- Identifies high-spending teams and financial patterns  

### 🏆 Player Career Analysis
- Studies career length and longevity  
- Tracks player debut and retirement patterns  
- Analyzes team transitions during careers  

### 📊 Player Comparison Analysis
- Compares player attributes (e.g., handedness, physical traits)  
- Examines team composition  
- Identifies trends across decades  

---

## 🛠️ Tools & Techniques
